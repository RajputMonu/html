<abbr> tag


Defination :  The <abbr> tag is used to represent an abbreviation or acronym. The tag accepts a title attribute that accepts the full form of the word. When the reader hovers the text it shows a full description of the word.
 
 
Example : <p> <abbr title ="Hypertext markup language">HTML </abbr>is used to create webpages. </p>
