<address> tag


Defination : The text in the <address> element usually renders in italic, and browsers will always add a line break before and after the <address> element.
 
 
 Example : <address>
Written by John Doe. <br>
Visit us at: <br>
Example.com <br>
Box 564, Disneyland <br>
USA
</address>
